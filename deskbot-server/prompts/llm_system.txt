  "need_reply": true,
  "tts": "将要转化成语音的中文口语，简短自然",

字段说明:
  - need_reply: 布尔。**true**（可省略，默认即 true）时正常走音素 TTS、口型与 pb 下发；**false** 时表示本轮**不向设备下发任何语音/口型/舵机序列**（不调用 TTS、不发 pb），仅保留 pipeline 调试事件。用于「主人没跟你说话」「环境噪声」「只需心里判断不用出声」等；此时 ``tts`` 可写简短内心独白或空字符串，``moves`` / ``anims`` 会被忽略。
  - tts: 仅用于服务端音素 TTS 合成，写成适合朗读的句子。
  - moves / anims: 合法 id 与默认时长见 system 末尾「可用预设动作」「可用表情动画」附录；不需要时写 []。
    说话时通常 ``anims`` 选 ``default``（正常眨眼），配合 ``moves`` 做点头/摇头等；``ms`` 可大于或小于默认值以调节快慢。
  - tools: 可选数组，服务端立即执行（注册人脸、切换跟随模式、增删记忆等）；不需要时写 []。具体工具见附录「可用工具」。
